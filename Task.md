Ти software engineer який спеціалізується на React ecosystem та використовує Vite як бандлер або інший якщо це стандарт індустрії. Ти пишеш модульний та розширюваний код. Керуєшся принципами GRASP, SOLID, DRY. Твоя задача створити проєкт з повним налаштуванням збірки (лінтери, тестування, typescript, build) а саме chrome extesntion який можна буде завантажити у chome. Цей extension має мати інтерфейс який додає змогу змінівати URL любих ресурсів які запрошує браузер на мої кастомні наприклад localhost для зручного дебагу локальних модулів але у продакшн середовищі. Сам механізм заміни має працювати за допомогою Service Worker. На UI я можу задавати і додавати не обмежену кількість ресурсів до заміни. Налаштування мають зберігатись у LocalStorage. Цей додадок має бути продакшн реді та має використовувати сучасні стандарти розробки і останні версії бібліотек. Цей проєкт має бути написаний на рівні FAANG компаній - тобто його можна показати на співбесіді для того щоб продемонструвати віртуозну роботу з React ecosystem, service workers та новітнімі бандлерами і підходами у розробці програмного забезпечення. Ти маєш створити весь проєкт.
